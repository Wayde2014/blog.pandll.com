计算机网络和因特网

物理媒体(传输介质)

  • 双绞铜线

    双绞线由两根隔离的铜线组成，每根大约1mm粗，以规则的螺旋形式排列。这两根线被绞合起来，以减少来自邻近类似的双绞线的电气干扰。多根双绞线捆扎形成一根电缆，双绞线外覆盖保护性防护层。

    最便宜、使用最普遍，多用于局域网LAN，速率从10Mbps到10Gbps。

  • 同轴电缆

    由两个同心铜导体组成。

  • 光纤

    光纤是光导纤维的简写，是一种由玻璃或塑料制成的纤维，可作为光传导工具。

    高传输速率，不受电磁干扰，信号衰减极低，难窃听。但是成本高，多用于主干网络。

  • 陆地无线电通道

    无线电信道承载电磁频谱中的信号。

    • 短距离

      蓝牙

    • 局域

      wifi

    • 广域

      运营商提供的蜂窝网络

  • 卫星无线电通道

    同步卫星、近地卫星

    高延时

分组交换中的时延、丢包和Traceroute原理

时延类型

  • 处理时延
  • 排队时延
  • 传输时延
  • 传播时延

丢包

链路队列容量有限，后到达分组没有地方存储，路由器将其丢弃。

Traceroute原理

往指定目的地发送多个特殊分组，当这些分组向着目的地传送时，经过的路由器接收到这些特殊分组之一时，路由器向源回送一个短报文，该报文包括该路由器名字和地址。

协议层次及其服务模型

  • 因特网协议栈

    • 应用层

      报文，HTTP、SMTP、FTP

    • 传输层

      报文段，TCP、UDP

    • 网络层

      数据报，IP

    • 链路层

      数据帧，以太网、WiFi、电缆接入网的DOCSIS协议

    • 物理层

      比特，双绞铜线、单模光纤

  • OCI模型

    ISO-国际标准组织提出，开放系统互联(OSI)模型

网络攻击

  • 病毒

    需要某种形式的用户交互触发

  • 蠕虫

    无需任何明显交互

  • 拒绝服务攻击(Denial-of-Service)

    • 弱点攻击

      软件或系统漏洞

    • 带宽洪泛

      发送大量数据，造成网络拥塞

    • 连接洪泛

      创建大量TCP连接，消耗连接数

    网络窃听
